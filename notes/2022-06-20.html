Functional programming

  • In Maths a function from a set A to a set B assignees to each element in A an element in B
    • Function is total if for every input in A will return an output in B
    • Function is partial if exists an input in A will not return an output in B
  • Functor is a type that has a and_then function
  • Monad is a type with a flat_map function
